Vous cherchez à signaler un joueur ? Veuillez prendre en compte les informations suivantes.

Preuve vidéo généralement requise

Pour presque toutes les infractions, nous ne bannirons les gens que si vous avez une preuve vidéo.

L'exception à cette règle peut être les infractions liées au chat (assurez-vous de fournir suffisamment de contexte).

Respecter la décision

Nous vous demandons de rester aimable, civil dans ces rapports.

Si le rapport est classé, c'est que les preuves n'étaient pas suffisantes.

Merci pour votre travail

Le personnel apprécie le temps que vous prenez pour enregistrer/poster des preuves vidéo des personnes qui enfreignent les règles.

Merci pour votre engagement à faire de {{ site.title }} un meilleur endroit !


Le processus de rapport est traité sur notre serveur Discord. Veuillez créer un compte si vous n'en avez pas encore un.
